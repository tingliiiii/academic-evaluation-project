## 🌊 完整工作流驗證

### 從請求到資料庫的完整鏈路

```
1️⃣ 用戶輸入
   ↓
2️⃣ 調用 GET /api/admin/wisdoms + /api/admin/tones
   ↓
3️⃣ 調用 POST /api/prompts/preview
   ↓ 
4️⃣ 生成 Prompt 模板（793 字符）
   ↓
5️⃣ 用戶確認後調用 POST /api/evaluations
   ↓
6️⃣ API 呼叫 Gemini API
   ↓
7️⃣ Gemini 生成評語（526 字符）
   ↓
8️⃣ API 在資料庫中建立:
   - Student 記錄（如不存在）
   - Evaluation 記錄
   - EvaluationWisdom 關聯
   ↓
9️⃣ 返回 201 Created + 評語 ID
   ↓
🔟 驗證：GET /api/evaluations/[id] → 200 ✅
```
